11.1

¿Qué es un Arreglo Unidimensional?

Un arreglo (array) es una colección de elementos del mismo tipo almacenados en posiciones contiguas de memoria. Cada elemento se accede mediante un índice entero que empieza en 0. Son la estructura de datos más fundamental: permiten almacenar y procesar decenas o miles de valores con una sola variable.

Definición formal

tipo nombre[tamaño]; — reserva tamaño posiciones contiguas de tipo tipo. Los índices válidos van de 0 a tamaño−1. Acceder al índice tamaño o mayor es comportamiento indefinido (error difícil de detectar).

Visualización de memoria — arreglo de 5 enteros

Índice01234
Variablenotas[0]notas[1]notas[2]notas[3]notas[4]
Valor ejemplo8.57.09.26.810.0
Dirección (relativa)0x000x080x100x180x20
⚠️ El índice empieza en 0, no en 1

En C++ el primer elemento es notas[0] y el último es notas[n-1]. El error off-by-one (escribir notas[n] creyendo que es el último) es uno de los bugs más frecuentes. El compilador no lo detecta; provoca corrupción de memoria silenciosa.

11.2

Declaración e Inicialización

C++ · formas de declarar e inicializar arreglos
declaracion_arrays.cpp 
// ① Declarar sin inicializar — valores basura (peligroso)
double notas[5];                   // 5 doubles con valores indeterminados

// ② Inicializar con lista de valores
double notas[5] = {8.5, 7.0, 9.2, 6.8, 10.0};

// ③ Inicializar todo en cero
int    contadores[10] = {};         // todos los elementos = 0

// ④ El compilador deduce el tamaño automáticamente
int    primos[] = {2, 3, 5, 7, 11};   // tamaño = 5 (deduce el compilador)

// ⑤ Tamaño constante — práctica recomendada
const int N = 30;
double calificaciones[N] = {};      // fácil de cambiar: solo editar N

// Acceso individual
notas[2] = 9.5;                       // modificar tercer elemento
cout << notas[0];                    // leer primer elemento
📌 Siempre usa constante para el tamaño

Declarar const int N = 30; y luego double arr[N]; hace el código mantenible: si el requisito cambia a 50 estudiantes, solo modificas una línea. Evita literales numéricos dispersos en el código (números mágicos).

11.3

Recorrido con for — El Patrón Fundamental

El bucle for con índice es el patrón canónico para procesar arreglos. La variable índice recorre desde 0 hasta n−1 inclusive. Todo algoritmo de arreglos se construye sobre este patrón.

C++ · ingresar, mostrar y sumar — tres recorridos fundamentales
recorrido.cpp 
const int N = 5;
double notas[N];

// RECORRIDO 1: Ingresar datos
for (int i = 0; i < N; i++) {
    cout << "Nota [" << i+1 << "]: ";
    cin  >> notas[i];
}

// RECORRIDO 2: Mostrar tabla
cout << "
Índice  Nota
";
for (int i = 0; i < N; i++)
    cout << setw(6) << i << setw(8) << fixed << setprecision(1) << notas[i] << "
";

// RECORRIDO 3: Acumulador
double suma = 0;
for (int i = 0; i < N; i++)
    suma += notas[i];
cout << "Promedio: " << suma / N << endl;
11.4

Paso de Arreglos a Funciones — El Puntero Implícito

En C++, cuando pasas un arreglo a una función, no se copia: se pasa la dirección del primer elemento (array decay). Esto significa que la función trabaja sobre el arreglo original y puede modificarlo. Por eso no se necesita & — el arreglo ya es, esencialmente, una referencia.

Firma correcta — tamaño como parámetro separado 
void mostrar(double arr[], int n) {
  for(int i=0;i<n;i++)
    cout<<arr[i]<<" ";
}
Error frecuente — sizeof dentro de función 
void mostrar(double arr[]) {
  // sizeof(arr) da 8 (tamaño del puntero)
  // NO el tamaño del arreglo original
}
C++ · funciones que reciben arreglo + tamaño
funciones_array.cpp 
// arr[] es modificable: multiplica por 10 cada elemento
void escalar(double arr[], int n, double factor) {
    for (int i = 0; i < n; i++) arr[i] *= factor;
}

// const: la función garantiza que NO modifica arr
void imprimir(const double arr[], int n) {
    for (int i = 0; i < n; i++)
        cout << arr[i] << " ";
    cout << endl;
}

int main() {
    const int N = 4;
    double datos[N] = {1.0, 2.0, 3.0, 4.0};
    escalar(datos, N, 10);    // datos ahora = (10, 20, 30, 40)
    imprimir(datos, N);       // 10 20 30 40
    return 0;
}
11.5

Estadísticas: Mínimo, Máximo y Promedio

C++ · calcularEstadisticas — tres funciones independientes
estadisticas.cpp 
double calcularMin(const double arr[], int n) {
    double min = arr[0];
    for (int i = 1; i < n; i++)
        if (arr[i] < min) min = arr[i];
    return min;
}

double calcularMax(const double arr[], int n) {
    double max = arr[0];
    for (int i = 1; i < n; i++)
        if (arr[i] > max) max = arr[i];
    return max;
}

double calcularProm(const double arr[], int n) {
    double suma = 0;
    for (int i = 0; i < n; i++) suma += arr[i];
    return suma / n;
}

// Encontrar el índice del máximo (para reportar qué estudiante)
int indiceMax(const double arr[], int n) {
    int idx = 0;
    for (int i = 1; i < n; i++)
        if (arr[i] > arr[idx]) idx = i;
    return idx;
}
11.6

Búsqueda Lineal

La búsqueda lineal recorre el arreglo desde el inicio hasta encontrar el elemento o agotar todos los elementos. Funciona en arreglos no ordenados. Costo: O(n) — en el peor caso revisa todos los elementos.

C++ · búsqueda lineal — retorna índice o -1
busqueda_lineal.cpp 
// Retorna el índice del elemento buscado, o -1 si no existe
int buscarLineal(const double arr[], int n, double objetivo) {
    for (int i = 0; i < n; i++)
        if (arr[i] == objetivo) return i;   // encontrado
    return -1;                              // no encontrado
}

// Uso:
int pos = buscarLineal(notas, N, 10.0);
if (pos != -1)
    cout << "Nota 10 en índice " << pos << endl;
else
    cout << "Nota 10 no encontrada
";

11.7

Ordenamiento Burbuja

El algoritmo de burbuja compara pares de elementos adyacentes e intercambia los que están en el orden incorrecto. Repite este proceso n−1 veces. Cada pasada "sube" el elemento mayor hacia el final, como una burbuja de aire en agua. Costo: O(n²).

C++ · ordenamiento burbuja con flag de optimización
burbuja.cpp 
void burbuja(double arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        bool huboIntercambio = false;    // optimización: detectar arreglo ya ordenado
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {      // par en orden incorrecto
                double temp = arr[j];
                arr[j]     = arr[j + 1];
                arr[j + 1] = temp;
                huboIntercambio = true;
            }
        }
        if (!huboIntercambio) break;       // arreglo ya ordenado → salir
    }
}
