🎯 Problema central
La empresa eléctrica regional cobra según el consumo mensual en kWh: 0–100 kWh → $0.08/kWh, 101–300 kWh → $0.10/kWh, 301–500 kWh → $0.13/kWh, más de 500 kWh → $0.16/kWh. Si el cliente es de categoría industrial (ingreso como char: 'I'), se añade un cargo fijo de $12.00. Calcular y mostrar: consumo, tarifa aplicada, cargo fijo, subtotal, IVA (12%) y total a pagar.

Extensión con switch — Parejas
⏱ 100 – 135 minutos

Agrega al programa un menú con switch que permita elegir la acción antes de calcular.

C++ · menú a agregar
tarifa_menu.cpp 
int opcion;
cout << "=== SISTEMA ELÉCTRICO ===\n";
cout << "1. Calcular tarifa\n";
cout << "2. Ver tabla de tarifas\n";
cout << "3. Salir\n";
cout << "Opción: "; cin >> opcion;

switch (opcion) {
    case 1:
        // Aquí va todo el código de la Fase 3
        break;
    case 2:
        // Mostrar tabla de rangos y tarifas
        break;
    case 3:
        cout << "Hasta luego.\n";
        break;
    default:
        cout << "Opción inválida.\n";
}
✓
Implementa la opción 2
El case 2 debe mostrar una tabla formateada con los 4 rangos, sus tarifas y el cargo adicional industrial. Usa setw de <iomanip> para alinear columnas.
