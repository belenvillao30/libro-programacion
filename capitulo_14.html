14.1

El Tipo string en C++

En C++ el tipo string (de la biblioteca <string>) es un arreglo de caracteres dinámico con gestión automática de memoria. A diferencia del arreglo de caracteres de C (char[]), string sabe su propio tamaño, puede crecer automáticamente y provee métodos de manipulación seguros.

string es internamente un arreglo de char

Cada carácter ocupa un byte y tiene un índice de 0 a length()-1. El último carácter lógico es s[s.length()-1]. La clase gestiona el terminador nulo interno de forma transparente. El tipo size_t retornado por length() es un entero sin signo: al compararlo con int puede haber advertencias del compilador.

C++ · declaración, inicialización y acceso a caracteres
string_basico.cpp 
#include <iostream>
#include <string>
using namespace std;

int main() {
    string nombre = "ESPAM MFL";

    cout << nombre.length()  << endl;  // 9 — número de caracteres
    cout << nombre[0]        << endl;  // E — primer carácter
    cout << nombre.back()    << endl;  // L — último carácter
    cout << nombre.empty()   << endl;  // 0 (false) — no está vacío

    // Recorrer carácter a carácter
    for (int i = 0; i < (int)nombre.length(); i++)
        cout << nombre[i] << " ";   // E S P A M   M F L
    cout << endl;

    return 0;
}

string s = "Programación en C++";

📌 string::npos — valor centinela de find()

string::npos es el valor máximo de size_t (~4 mil millones). Cuando find() no encuentra la subcadena retorna string::npos. Siempre verifica antes de usar el índice retornado, o causarás acceso fuera de límites.

14.4

Procesamiento Carácter a Carácter

Las operaciones más potentes sobre cadenas recorren el string un carácter a la vez con un bucle for y aplican funciones de <cctype> a cada posición.

C++ · funciones de procesamiento de cadenas
procesamiento_texto.cpp 
#include <iostream>
#include <string>
#include <cctype>
using namespace std;

// Convertir toda la cadena a mayúsculas
string aMayusculas(string s) {
    for (int i=0; i<(int)s.length(); i++)
        s[i] = toupper(s[i]);
    return s;
}

// Contar vocales en la cadena
int contarVocales(const string& s) {
    int cnt = 0;
    for (int i=0; i<(int)s.length(); i++) {
        char c = tolower(s[i]);
        if (c=='a'||c=='e'||c=='i'||c=='o'||c=='u') cnt++;
    }
    return cnt;
}

// Invertir la cadena
string invertir(const string& s) {
    string inv = "";
    for (int i=(int)s.length()-1; i>=0; i--)
        inv += s[i];
    return inv;
}

int main() {
    string texto = "Hola Mundo";
    cout << aMayusculas(texto)  << endl;  // HOLA MUNDO
    cout << contarVocales(texto) << endl;  // 4
    cout << invertir(texto)      << endl;  // odnuM aloH
    return 0;
}
14.5

Validación de Texto con Funciones Reutilizables

La validación de entrada de texto es una de las aplicaciones más comunes del procesamiento de cadenas en sistemas reales: verificar que un campo solo contiene letras, que un código tiene el formato correcto, que una contraseña cumple requisitos mínimos de seguridad.

C++ · validadores reutilizables con const string&
validadores.cpp 
#include <string>
#include <cctype>
using namespace std;

// Solo letras y espacios (para nombres de personas)
bool soloLetras(const string& s) {
    if (s.empty()) return false;
    for (int i=0; i<(int)s.length(); i++)
        if (!isalpha(s[i]) && s[i] != ' ') return false;
    return true;
}

// Contraseña: mínimo 8 caracteres, al menos 1 dígito y 1 mayúscula
bool esPasswordValida(const string& pwd) {
    if ((int)pwd.length() < 8) return false;
    bool tieneDigito=false, tieneMayus=false;
    for (int i=0; i<(int)pwd.length(); i++) {
        if (isdigit(pwd[i]))  tieneDigito = true;
        if (isupper(pwd[i]))  tieneMayus  = true;
    }
    return tieneDigito && tieneMayus;
}

// Verificar palíndromo ignorando mayúsculas y espacios
bool esPalindromo(string s) {
    string limpia = "";
    for (int i=0; i<(int)s.length(); i++)
        if (isalpha(s[i])) limpia += tolower(s[i]);
    int ini=0, fin=(int)limpia.length()-1;
    while (ini<fin)
        if (limpia[ini++] != limpia[fin--]) return false;
    return true;
}  // "Anita lava la tina" → true
🤖 IA y el procesamiento de strings

La IA suele proponer regex (<regex>) para validación de texto: es potente pero compleja para principiantes. Para esta asignatura, las validaciones con cctype son más pedagógicas porque hacen explícito el recorrido carácter a carácter. Usa la solución con regex de la IA para contrastar, pero asegúrate de entender y poder reescribir la versión manual.
