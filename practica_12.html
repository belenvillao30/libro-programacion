Sistema de Notas por Asignatura — Matriz Estudiantes × Materias

arreglos 2D · bucles anidados · columnas constantes · promedio por fila y columna · mejor estudiante y materia más difícil

Duración180 min (3h)
ModalidadIndividual → Parejas
Entregable: .cpp + trazas + bitácora
F1 · Diseño F2 · Pseudocódigo F3 · C++ F4 · Extensión F5 · Análisis IA F6 · Bitácora
Objetivo de la Sesión
El estudiante implementa un sistema de registro de notas con una matriz de 5 estudiantes × 4 asignaturas, calcula promedios por estudiante (filas) y por asignatura (columnas), identifica el mejor estudiante y la materia con mayor dificultad, e imprime un reporte tabular alineado con formato profesional.
🎓 Contexto del Problema
La coordinación académica de ESPAM MFL necesita procesar las notas del primer parcial de 5 estudiantes en 4 asignaturas: Programación, Matemáticas, Física e Inglés. El sistema debe: 1) ingresar notas con validación [0–10] 2) calcular promedio por estudiante 3) calcular promedio por asignatura 4) identificar el mejor estudiante y la asignatura más baja 5) generar un reporte matricial completo.

Calcula a mano: promedio de cada estudiante, promedio de cada asignatura. Identifica quién tiene el promedio más alto y cuál asignatura tiene el promedio más bajo.

Firma de cada función
Diseña la firma de las 5 funciones del sistema: ingresar, promediosPorFila, promediosPorColumna, mostrarReporte, indiceMin. Para cada parámetro de arreglo/matriz decide si necesita const. Recuerda: las columnas deben ser constantes ([][4]) en las firmas.

Implementación en C++ — Individual
55 – 105 min
C++ · sistema de notas por asignatura — implementación completa
notas_asignaturas.cpp 
#include <iostream>
#include <iomanip>
#include <string>
using namespace std;

const int FILAS=5, COLS=4;
const string ASIGS[COLS] = {"Progr","Matem","Fisica","Ingles"};

void ingresar(double m[][COLS], int f);
void promediosPorFila(const double m[][COLS], int f, double pf[]);
void promediosPorCol(const double m[][COLS], int f, double pc[]);
int  indiceMax(const double arr[], int n);
int  indiceMin(const double arr[], int n);
void mostrarReporte(const double m[][COLS], int f,
                   const double pf[], const double pc[]);

int main() {
    double notas[FILAS][COLS];
    double promFila[FILAS], promCol[COLS];
    ingresar(notas, FILAS);
    promediosPorFila(notas, FILAS, promFila);
    promediosPorCol(notas, FILAS, promCol);
    mostrarReporte(notas, FILAS, promFila, promCol);
    cout<<"
Mejor estudiante : E"<<indiceMax(promFila,FILAS)+1<<"
";
    cout<<"Materia más baja : "<<ASIGS[indiceMin(promCol,COLS)]<<"
";
    return 0;
}

void ingresar(double m[][COLS], int f) {
    for(int i=0;i<f;i++)
        for(int j=0;j<COLS;j++) {
            do {
                cout<<"  E"<<i+1<<" "<<ASIGS[j]<<": ";
                cin>>m[i][j];
            } while(m[i][j]<0||m[i][j]>10);
        }
}
void promediosPorFila(const double m[][COLS],int f,double pf[]) {
    for(int i=0;i<f;i++){double s=0;for(int j=0;j<COLS;j++)s+=m[i][j];pf[i]=s/COLS;}
}
void promediosPorCol(const double m[][COLS],int f,double pc[]) {
    for(int j=0;j<COLS;j++){double s=0;for(int i=0;i<f;i++)s+=m[i][j];pc[j]=s/f;}
}
int indiceMax(const double a[],int n){int m=0;for(int i=1;i<n;i++)if(a[i]>a[m])m=i;return m;}
int indiceMin(const double a[],int n){int m=0;for(int i=1;i<n;i++)if(a[i]<a[m])m=i;return m;}
void mostrarReporte(const double m[][COLS],int f,
                   const double pf[],const double pc[]) {
    cout<<"
       ";
    for(int j=0;j<COLS;j++) cout<<setw(8)<<ASIGS[j];
    cout<<setw(9)<<"Prom
";
    for(int i=0;i<f;i++){
        cout<<"  E"<<i+1<<"  ";
        for(int j=0;j<COLS;j++)
            cout<<fixed<<setprecision(1)<<setw(8)<<m[i][j];
        cout<<setw(9)<<pf[i]<<"
";
    }
    cout<<"  Prom  ";
    for(int j=0;j<COLS;j++) cout<<setw(8)<<pc[j];
    cout<<"
";
}
