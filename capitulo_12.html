12.1

Estructura Bidimensional — Filas y Columnas

Una matriz es un arreglo de filas × columnas: cada elemento se identifica con dos índices mat[fila][columna]. Modela naturalmente tablas, imágenes de píxeles, tableros de juego, y datos como notas por asignatura o inventarios por sucursal.

Modelo mental: hoja de cálculo

El primer índice es la fila (el estudiante) y el segundo es la columna (la asignatura). mat[2][1] es la nota del tercer estudiante en la segunda asignatura. En C++ las filas se almacenan consecutivamente en memoria (row-major order): primero toda la fila 0, luego la fila 1, etc.

12.4

Paso de Matrices a Funciones — Columnas Constantes

Al pasar una matriz a una función, el número de columnas debe ser constante conocido en compilación. C++ necesita saber las columnas para calcular la dirección de cada elemento.

Correcto — columnas fijas 
const int COLS = 4;
void mostrar(const int m[][COLS],
             int filas) {
  // COLS conocido al compilar ✓
}
Error — ambas dimensiones variables 
// No compila en C++ clásico:
void mostrar(int m[][], int f, int c) {
  // El compilador no puede
  // calcular la dirección de m[i][j]
}
C++ · funciones con firma correcta
funciones_matriz.cpp 
const int COLS = 4;

void mostrarMatriz(const int m[][COLS], int filas) {
    for(int i=0;i<filas;i++) {
        for(int j=0;j<COLS;j++) cout<<setw(5)<<m[i][j];
        cout<<"
";
    }
}

int sumaTotal(const int m[][COLS], int filas) {
    int s=0;
    for(int i=0;i<filas;i++)
        for(int j=0;j<COLS;j++) s+=m[i][j];
    return s;
}
12.5

Suma de Matrices y Transpuesta

C++ · suma elemento a elemento + transposición
operaciones_matriz.cpp 
const int N = 3;

// Suma: C[i][j] = A[i][j] + B[i][j]
void sumarMatrices(const int A[][N], const int B[][N],
                   int C[][N], int n) {
    for(int i=0;i<n;i++)
        for(int j=0;j<n;j++)
            C[i][j] = A[i][j] + B[i][j];
}

// Transpuesta: T[j][i] = A[i][j]  (intercambiar índices)
void transponerMatriz(const int A[][N], int T[][N], int n) {
    for(int i=0;i<n;i++)
        for(int j=0;j<n;j++)
            T[j][i] = A[i][j];
}

Diagonales y Traza de una Matriz Cuadrada

C++ · traza y diagonal secundaria
diagonales.cpp 
const int N=3;

// Traza = suma diagonal principal m[i][i]
int traza(const int m[][N], int n) {
    int s=0;
    for(int i=0;i<n;i++) s+=m[i][i];
    return s;
}
// Diagonal secundaria: m[i][n-1-i]
int trazaSecundaria(const int m[][N], int n) {
    int s=0;
    for(int i=0;i<n;i++) s+=m[i][n-1-i];
    return s;
}
// Simetría: m[i][j] == m[j][i] para todo i,j
bool esSimétrica(const int m[][N], int n) {
    for(int i=0;i<n;i++)
        for(int j=i+1;j<n;j++)
            if(m[i][j]!=m[j][i]) return false;
    return true;
}
🤖 IA y las matrices

La IA frecuentemente propone vector<vector<int>> para matrices, eliminando la restricción de columnas fijas. Esta solución moderna es más flexible pero tiene mayor costo de acceso en caché. Usamos arreglos 2D clásicos primero para entender la mecánica de memoria contigua; ese conocimiento hace que la transición a vector sea natural y bien comprendida.
