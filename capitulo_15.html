Resultado de Aprendizaje — Capítulo 15 (Integrador)
El estudiante diseña, implementa y documenta un sistema de software modular en C++ que integra todos los conceptos del curso: funciones con paso por valor y referencia, arreglos unidimensionales y matrices, ordenamiento, búsqueda binaria, procesamiento de cadenas y uso crítico de IA como herramienta asistida. 15.1

Visión del Proyecto — ¿Qué Construiremos?

A lo largo del curso aprendiste las piezas fundamentales de la programación: algoritmos, variables, estructuras de control, funciones, arreglos y cadenas. El capítulo 15 es el momento de ensamblar todas esas piezas en un sistema coherente que resuelva un problema real. No es un ejercicio más: es el producto final de tu aprendizaje.

Sistema de Gestión Académica — ESPAM MFL

Un sistema de consola en C++ que permite registrar hasta 30 estudiantes, gestionar sus 4 notas por materia, calcular promedios, clasificar el rendimiento, ordenar por nota o nombre, buscar por cédula y generar reportes estadísticos. Cada módulo es un conjunto de funciones cohesivas que no usa variables globales.

Mapa de funcionalidades del sistema

MóduloFuncionalidadesConceptos del curso
EstudiantesRegistrar, buscar por cédula, listar, eliminar lógicoArreglos paralelos, strings, búsqueda lineal/binaria
NotasIngresar 4 notas, calcular promedio, clasificar (A/B/C/D/F)Arreglos 2D, funciones, paso por referencia
OrdenamientoOrdenar por promedio descendente o por nombre A-ZBurbuja/selección con arreglos paralelos

MenúMenú principal con do-while + switch, submenús por móduloEstructuras de control anidadas
📌 Principio rector: main() solo orquesta

El main() del proyecto integrador no realiza ningún cálculo directo. Solo declara las estructuras de datos, muestra el menú y llama funciones. Todo el procesamiento vive en funciones con nombres descriptivos. Este principio —que has practicado desde el Cap. 8— es la base del software profesional modular.

📌 Por qué arreglos paralelos y no struct

Un struct Estudiante sería más elegante en C++ moderno, pero requiere conocimiento que no forma parte del currículo de este curso. Los arreglos paralelos son una solución completamente válida y demuestran el mismo principio: todos los datos del índice i pertenecen al mismo estudiante. El convenio debe ser explícito en comentarios.

15.3

Módulo Estudiantes — Registro, Búsqueda y Eliminación

C++ · Módulo completo de gestión de estudiantes
modulo_estudiantes.cpp 
// ── Validar cédula: exactamente 10 caracteres numéricos ──────────
bool cedulaValida(const string& ced) {
    if (ced.length() != 10) return false;
    for (char c : ced)
        if (!isdigit(c)) return false;
    return true;
}

string leerCedula(const string& msg) {
    string ced;
    do {
        cout << msg; cin >> ced;
        if (!cedulaValida(ced))
            cout << "  ✗ Cédula inválida (10 dígitos).
";
    } while (!cedulaValida(ced));
    return ced;
}

// ── Búsqueda lineal por cédula entre activos ─────────────────────
int buscarPorCedula(const string& cedula) {
    for (int i = 0; i < totalEst; i++)
        if (activo[i] && cedulas[i] == cedula) return i;
    return -1;   // no encontrado
}

// ── Registrar nuevo estudiante ────────────────────────────────────
bool registrarEstudiante(const string& cedula, const string& nombre) {
    if (totalEst >= MAX_EST) {
        cout << "  ✗ Sistema lleno (máximo " << MAX_EST << " estudiantes).
";
        return false;
    }
    if (buscarPorCedula(cedula) != -1) {
        cout << "  ✗ Cédula ya registrada.
";
        return false;
    }
    cedulas  [totalEst] = cedula;
    nombres  [totalEst] = nombre;
    promedios[totalEst] = 0.0;
    activo   [totalEst] = true;
    for (int j = 0; j < NUM_NOTAS; j++) notas[totalEst][j] = 0.0;
    totalEst++;
    cout << "  ✔ Estudiante registrado correctamente.
";
    return true;
}

// ── Listar todos los activos ──────────────────────────────────────
void listarEstudiantes() {
    separador("LISTADO DE ESTUDIANTES");
    int count = 0;
    cout << left << setw(4) << "#"
         << setw(12) << "Cédula"
         << setw(28) << "Nombre"
         << setw(8)  << "Prom"
         << "Estado
";
    cout << string(60, '-') << "
";
    for (int i = 0; i < totalEst; i++) {
        if (!activo[i]) continue;
        count++;
        cout << left
             << setw(4)  << count
             << setw(12) << cedulas[i]
             << setw(28) << nombres[i]
             << fixed << setprecision(2)
             << setw(8)  << promedios[i]
             << clasificarRendimiento(promedios[i]) << "
";
    }
    if (count == 0) cout << "  (sin registros)
";
    else cout << "
Total activos: " << count << "
";
}

// ── Eliminación lógica (no borra datos, marca inactivo) ──────────
bool eliminarEstudiante(const string& cedula) {
    int idx = buscarPorCedula(cedula);
    if (idx == -1) { cout << "  ✗ Cédula no encontrada.
"; return false; }
    activo[idx] = false;
    cout << "  ✔ Estudiante " << nombres[idx] << " eliminado del sistema.
";
    return true;
}

Módulo Notas — Ingreso, Cálculo y Clasificación

C++ · Módulo de notas — validación, promedio y clasificación
modulo_notas.cpp 
double leerNota(const string& msg) {
    double n;
    do {
        cout << msg; cin >> n;
        if (n < 0 || n > 10)
            cout << "  ✗ Nota fuera de rango [0, 10].
";
    } while (n < 0 || n > 10);
    return n;
}

bool ingresarNotas(const string& cedula) {
    int idx = buscarPorCedula(cedula);
    if (idx == -1) {
        cout << "  ✗ Estudiante no encontrado.
";
        return false;
    }
    cout << "  Ingresando notas para: " << nombres[idx] << "
";
    for (int j = 0; j < NUM_NOTAS; j++) {
        string msg = "  Nota " + to_string(j+1) + " [0-10]: ";
        notas[idx][j] = leerNota(msg);
    }
    promedios[idx] = calcularPromedio(idx);
    cout << "  ✔ Promedio calculado: "
         << fixed << setprecision(2) << promedios[idx] << "  → "
         << clasificarRendimiento(promedios[idx]) << "
";
    return true;
}

double calcularPromedio(int idx) {
    double suma = 0;
    for (int j = 0; j < NUM_NOTAS; j++) suma += notas[idx][j];
    return suma / NUM_NOTAS;
}

string clasificarRendimiento(double prom) {
    if      (prom >= 9.0) return "A  (Sobresaliente)";
    else if (prom >= 8.0) return "B  (Muy Bueno)";
    else if (prom >= 7.0) return "C  (Bueno)";
    else if (prom >= 5.0) return "D  (Deficiente)";
    else                   return "F  (Reprobado)";
}

void recalcularTodosPromedios() {
    for (int i = 0; i < totalEst; i++)
        if (activo[i]) promedios[i] = calcularPromedio(i);
}

Módulo Reportes — Estadísticas y Visualización ASCII

C++ · Módulo de reportes — estadístico, top 3 e histograma
modulo_reportes.cpp 
void reporteEstadistico() {
    separador("REPORTE ESTADÍSTICO");
    int    activos = 0, aprobados = 0;
    double suma = 0, minP = 10, maxP = 0;
    for (int i = 0; i < totalEst; i++) {
        if (!activo[i]) continue;
        activos++;
        suma += promedios[i];
        if (promedios[i] < minP) minP = promedios[i];
        if (promedios[i] > maxP) maxP = promedios[i];
        if (promedios[i] >= 7.0) aprobados++;
    }
    if (activos == 0) { cout << "  (sin datos)
"; return; }
    cout << fixed << setprecision(2)
         << "  Estudiantes activos : " << activos    << "
"
         << "  Promedio general    : " << suma/activos << "
"
         << "  Nota mínima         : " << minP         << "
"
         << "  Nota máxima         : " << maxP         << "
"
         << "  Aprobados (≥7)      : " << aprobados
         << "  (" << setprecision(1) << (100.0*aprobados/activos) << "%)
";
}

void reporteTop3() {
    separador("TOP 3 ESTUDIANTES");
    // Ordenar copia temporal sin tocar el arreglo original
    int indices[MAX_EST], n = 0;
    for (int i = 0; i < totalEst; i++) if (activo[i]) indices[n++] = i;
    // Burbuja sobre índices
    for (int i = 0; i < n-1; i++)
        for (int j = 0; j < n-i-1; j++)
            if (promedios[indices[j]] < promedios[indices[j+1]])
                swap(indices[j], indices[j+1]);
    int mostrar = min(n, 3);
    for (int k = 0; k < mostrar; k++) {
        int i = indices[k];
        cout << "  " << (k+1) << ". " << left << setw(28) << nombres[i]
             << fixed << setprecision(2) << promedios[i] << "  "
             << clasificarRendimiento(promedios[i]) << "
";
    }
}

void histogramaRendimiento() {
    separador("HISTOGRAMA DE RENDIMIENTO");
    int cA=0, cB=0, cC=0, cD=0, cF=0;
    for (int i = 0; i < totalEst; i++) {
        if (!activo[i]) continue;
        double p = promedios[i];
        if      (p >= 9) cA++;
        else if (p >= 8) cB++;
        else if (p >= 7) cC++;
        else if (p >= 5) cD++;
        else            cF++;
    }
    auto barra = [](int n){ return string(n, '█'); };
    cout << "  A (≥9.0): " << barra(cA) << " (" << cA << ")
"
         << "  B (≥8.0): " << barra(cB) << " (" << cB << ")
"
         << "  C (≥7.0): " << barra(cC) << " (" << cC << ")
"
         << "  D (≥5.0): " << barra(cD) << " (" << cD << ")
"
         << "  F (<5.0): " << barra(cF) << " (" << cF << ")
";
}

Integración — Módulo de Ordenamiento y Menú Principal

C++ · Ordenamiento de arreglos paralelos + menú principal
modulo_ordenamiento_main.cpp 
// ── Intercambiar TODOS los arreglos paralelos en posición i y j ──
void intercambiarEstudiantes(int i, int j) {
    swap(cedulas[i],   cedulas[j]);
    swap(nombres[i],   nombres[j]);
    swap(promedios[i], promedios[j]);
    swap(activo[i],    activo[j]);
    for (int k = 0; k < NUM_NOTAS; k++) swap(notas[i][k], notas[j][k]);
}

// ── Ordenar por promedio descendente (burbuja) ───────────────────
void ordenarPorPromedio() {
    for (int i = 0; i < totalEst - 1; i++)
        for (int j = 0; j < totalEst - i - 1; j++)
            if (promedios[j] < promedios[j+1])
                intercambiarEstudiantes(j, j+1);
    cout << "  ✔ Ordenado por promedio descendente.
";
}

// ── Ordenar por nombre A-Z (selección) ──────────────────────────
void ordenarPorNombre() {
    for (int i = 0; i < totalEst - 1; i++) {
        int minIdx = i;
        for (int j = i+1; j < totalEst; j++)
            if (nombres[j] < nombres[minIdx]) minIdx = j;
        if (minIdx != i) intercambiarEstudiantes(i, minIdx);
    }
    cout << "  ✔ Ordenado por nombre A-Z.
";
}

// ── Menú principal ────────────────────────────────────────────────
int main() {
    // Inicializar arreglo activo
    for (int i = 0; i < MAX_EST; i++) activo[i] = false;

    int opcion;
    do {
        mostrarMenuPrincipal();
        cin >> opcion;
        switch (opcion) {
            case 1: { // Registrar
                string ced = leerCedula("  Cédula: ");
                cin.ignore();
                string nom;
                cout << "  Nombre completo: ";
                getline(cin, nom);
                registrarEstudiante(ced, nom); break; }
            case 2: { ingresarNotas(leerCedula("  Cédula: ")); break; }
            case 3: listarEstudiantes(); break;
            case 4: fichaEstudiante(leerCedula("  Cédula: ")); break;
            case 5: eliminarEstudiante(leerCedula("  Cédula: ")); break;
            case 6: ordenarPorPromedio(); break;
            case 7: ordenarPorNombre(); break;
            case 8: { // Sub-menú reportes
                int r;
                do {
                    mostrarMenuReportes();
                    cin >> r;
                    if      (r==1) reporteEstadistico();
                    else if (r==2) reporteTop3();
                    else if (r==3) histogramaRendimiento();
                } while (r != 4);
                break; }
            case 9: cout << "Sistema cerrado. ¡Hasta pronto!
"; break;
            default: cout << "  ✗ Opción inválida.
";
        }
    } while (opcion != 9);
    return 0;
}
⚠️ Arreglos paralelos: la consistencia es tu responsabilidad

Cada vez que muevas o elimines un estudiante, debes mover todos los arreglos paralelos en la misma operación. La función intercambiarEstudiantes(i, j) encapsula esta responsabilidad para que no puedas olvidar ningún arreglo. Es un patrón de diseño: agrupar operaciones que deben ir siempre juntas en una sola función.

15.7

Estrategia de Uso de IA en el Proyecto Integrador

🤖 Cómo usar la IA en un proyecto de 300+ líneas — protocolo profesional

Un proyecto integrador es el escenario donde la IA puede ayudar genuinamente y donde también puede hacerte fallar si la usas mal. Protocolo recomendado en tres pasos: (1) Diseño propio: escribe tú los prototipos y la arquitectura de módulos antes de consultar a la IA. Si empiezas por la IA, obtienes código que no entiendes y no puedes defender. (2) IA para funciones acotadas: pide a la IA que implemente una función específica con firma exacta que tú especifiques. Nunca pidas "el sistema completo". (3) Revisión y prueba obligatoria: toda función generada por IA debe pasar por tus casos de prueba manuales y ser explicable por ti línea a línea. Si no puedes explicarla, no la entregues.

Solicitud a la IACalidadRazón
"Implementa bool cedulaValida(const string& ced) que verifica 10 dígitos numéricos"✅ ExcelenteFirma y comportamiento completamente especificados
"Genera el reporte estadístico usando mis arreglos globales nombres[], promedios[], activo[], totalEst"⚠️ Usar con cuidadoPuede funcionar, pero debes verificar cada cálculo
