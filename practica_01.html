ModalidadMixto: individual + parejas

🎯 Problema central de la sesión
Una tienda necesita calcular el precio final con descuento e impuesto de un producto. El programa debe solicitar el precio base, el porcentaje de descuento y el porcentaje de impuesto (IVA), y mostrar: precio con descuento, monto del impuesto y precio final a pagar.

Fórmulas:
· precio_descuento = precio_base × (1 − descuento/100)
· impuesto = precio_descuento × (iva/100)
· precio_final = precio_descuento + impuesto

F3
Implementación en C++ — Individual
⏱ 50 – 90 minutos

Traduce el pseudocódigo verificado a C++. Cada línea debe tener un comentario que explique su propósito — esto se evalúa en la defensa técnica.

C++ · Referencia
precio_final.cpp 
#include <iostream>
#include <iomanip>    // Para setprecision y fixed
using namespace std;

int main() {
    // --- Variables de entrada ---
    double precio_base, descuento, iva;

    // --- Variables de salida ---
    double precio_desc, monto_imp, precio_final;

    // --- Entrada de datos ---
    cout << "Ingrese el precio base: $";
    cin  >> precio_base;

    cout << "Ingrese el descuento (%): ";
    cin  >> descuento;

    cout << "Ingrese el IVA (%): ";
    cin  >> iva;

    // --- Proceso ---
    precio_desc  = precio_base * (1.0 - descuento / 100.0);
    monto_imp    = precio_desc  * (iva / 100.0);
    precio_final = precio_desc  + monto_imp;

    // --- Salida con 2 decimales ---
    cout << fixed << setprecision(2);
    cout << "\n--- Resumen de compra ---\n";
    cout << "Precio con descuento: $" << precio_desc  << endl;
    cout << "Monto de impuesto:    $" << monto_imp    << endl;
    cout << "Precio final a pagar: $" << precio_final << endl;

    return 0;
}
✓
Pruebas de verificación
Prueba con tres conjuntos de valores distintos y registra los resultados. Si el programa no compila, identifica el mensaje de error y corrígelo antes de pedir ayuda.
F4
Revisión cruzada en parejas
⏱ 90 – 130 minutos

Intercambia tu programa con un compañero. Debes revisar el código ajeno y documentar tus hallazgos.

A
¿El código compila sin errores?
Compila el código del compañero. Si hay errores, identifícalos y anótalos sin corregirlos todavía.
B
¿Las fórmulas son correctas?
Verifica que las tres fórmulas coincidan con el enunciado. Anota cualquier discrepancia.
C
¿Los comentarios son descriptivos?
¿Los comentarios explican por qué o solo repiten qué hace la línea? Propón una mejora para al menos un comentario.
D
Devolución de retroalimentación
Devuelve el trabajo con tus notas escritas. El autor original lee la retroalimentación y decide si aplica los cambios o justifica por qué no.
F5
Análisis crítico con IA — Grupal
⏱ 130 – 160 minutos
🤖 Actividad documentada de uso de IA

Esta es la única fase de la sesión donde se permite usar IA. Todo uso debe quedar registrado en tu bitácora con el prompt exacto utilizado y el análisis de la respuesta.

1
Pide a una IA una solución alternativa al mismo problema
Usa el prompt: "Escribe un programa en C++ que calcule el precio final de un producto aplicando descuento e IVA." — sin dar más contexto.
2
Compara las dos soluciones
¿Qué diferencias encuentras en los nombres de variables, los mensajes de salida, el uso de setprecision? ¿La IA incluyó comentarios? ¿Son más o menos descriptivos que los tuyos?
3
¿Hay errores en la solución de la IA?
Ejecuta el código generado por la IA. ¿Compila? ¿Los resultados son correctos con tus mismos valores de prueba? Documenta cada error que encuentres.
