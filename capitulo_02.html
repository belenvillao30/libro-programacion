2.1

Concepto de Dato y Tipo de Dato

Antes de escribir una variable es indispensable entender qué es un dato y por qué el tipo importa. Un dato es cualquier objeto manipulable por el programa: un número ingresado por el usuario, un carácter leído del teclado, un valor calculado en memoria.

Definición formal

Un tipo de dato es una clasificación que determina: (1) el conjunto de valores posibles para ese dato, (2) el espacio de memoria que ocupa, y (3) las operaciones permitidas sobre él. Declarar el tipo incorrecto no es un error de estilo — es un error lógico que produce resultados incorrectos.

A nivel de máquina, todos los datos son secuencias de bits. El tipo le dice al compilador cómo interpretar esos bits: los mismos 32 bits pueden ser un entero, un número real o cuatro caracteres, dependiendo del tipo declarado.

🤖 Reflexión sobre IA

Cuando la IA genera código, elige los tipos de forma automática — a veces correctamente, a veces no. Un tipo incorrecto produce resultados silenciosamente erróneos: por ejemplo, dividir dos int en C++ descarta los decimales sin avisar. Si no entiendes los tipos, no puedes detectar estos errores en el código que la IA te genera.

2.2

Tipos de Datos Fundamentales en C++

C++ tiene tipos primitivos incorporados. Para esta unidad trabajaremos con los siguientes cinco grupos:

Enteros

Números sin parte decimal. Pueden ser positivos, negativos o cero.

int short long long long

int: −2,147,483,647 a 2,147,483,647 (32 bits)

Flotantes

Números con parte decimal. double es el estándar del curso por su mayor precisión.

float double long double

double: ~15 dígitos significativos (64 bits)

Caracteres

Un solo símbolo del conjunto ASCII. Se escribe entre comillas simples.

char

Ejemplo: 'A', '7', '+', ' '

Booleanos

Solo dos valores posibles: verdadero o falso. Base de toda condición.

bool

Valores: true (1) o false (0)

Cadenas de texto

Secuencia de caracteres. Requiere la librería <string>. Se escribe entre comillas dobles.

string

Ejemplo: "Hola", "ESPAM MFL", "123 abc" (los dígitos como cadena no son operables aritméticamente)

⚠️ float vs double

Usa siempre double para cálculos con decimales en este curso. float tiene solo ~7 dígitos de precisión y puede acumular errores de redondeo en operaciones encadenadas. La diferencia se nota en resultados financieros o científicos.

2.3

Variables: Declaración, Inicialización y Reglas de Nombres

Una variable es un espacio en memoria con un nombre que permite almacenar y modificar un dato durante la ejecución del programa. La analogía clásica es una caja etiquetada: la etiqueta es el nombre, el contenido es el valor, y el tamaño de la caja depende del tipo.

2.4

Constantes: const y #define

Una constante almacena un valor que no puede cambiar durante la ejecución. Usarlas en lugar de "números mágicos" hace el código más legible y fácil de mantener.

C++
constantes.cpp 
#include <iostream>
using namespace std;

// Opción 1: const — tiene tipo, es verificada por el compilador ✅
const double IVA       = 0.12;
const double PI        = 3.14159265;
const int    MAX_NOTAS = 10;

// Opción 2: #define — sin tipo, reemplazo textual (evitar en C++ moderno)
// #define IVA 0.12  ← no recomendado: no tiene tipo ni alcance

int main() {
    double precio = 100.0;
    double impuesto = precio * IVA;   // Claro y mantenible
    double total    = precio + impuesto;

    cout << "Impuesto: $" << impuesto << endl;
    cout << "Total:    $" << total    << endl;

    // IVA = 0.15; ← ERROR: no se puede modificar una constante
    return 0;
}
📌 Convención de nombres para constantes

Por convención, las constantes se escriben en MAYÚSCULAS_CON_GUION_BAJO (IVA, MAX_NOTAS, PRECIO_BASE). Esto las distingue visualmente de las variables en el código.

2.5

Operadores y Expresiones Aritméticas

Una expresión es una combinación de valores, variables y operadores que produce un resultado. Comprender la precedencia y el efecto del tipo en las operaciones es esencial para evitar errores silenciosos.

⚠️ El error más común — División entera

int a = 7, b = 2; double r = a / b; — el resultado es 3.0, no 3.5. La división ocurre entre enteros antes de asignarse al double. Para obtener 3.5: double r = (double)a / b; o double r = a / 2.0;

2.6

Salida Formateada y Secuencias de Escape

Un programa no solo debe calcular bien — debe mostrar los resultados de forma clara y legible. C++ ofrece herramientas de formato en <iomanip>.

Secuencias de escape en cout

SecuenciaEfectoUso típico
\nNueva líneaSeparar bloques de salida
\tTabulación horizontalAlinear columnas
\\Barra invertida literalRutas de archivo
\"Comilla doble literalMostrar texto entre comillas
\0Carácter nuloFin de cadenas estilo C

Formato de decimales con <iomanip>

C++
formato.cpp 
#include <iostream>
#include <iomanip>   // setprecision, fixed, setw
using namespace std;

int main() {
    double precio = 1234.5678;

    // Sin formato
    cout << precio << endl;               // → 1234.57

    // Con fixed y 2 decimales (el más usado en el curso)
    cout << fixed << setprecision(2);
    cout << precio << endl;               // → 1234.57

    // Con 4 decimales
    cout << setprecision(4);
    cout << precio << endl;               // → 1234.5678

    // Alineación con setw (ancho de campo)
    cout << fixed << setprecision(2);
    cout << "Precio: $" << setw(10) << precio << endl;
    cout << "Total:  $" << setw(10) << precio * 1.12 << endl;

    // Secuencias de escape
    cout << "Nombre\tApellido\tNota\n";
    cout << "Ana\tGarcía\t\t9.5\n";

    return 0;
}
📌 fixed + setprecision

fixed activa el modo de notación decimal fija y se mantiene activo para todas las salidas siguientes. setprecision(2) fija el número de decimales. Úsalos juntos siempre que muestres precios, notas o medidas.

2.7

Conversión de Tipos: Implícita y Explícita

Cuando una operación involucra tipos distintos, C++ convierte automáticamente el tipo "menor" al "mayor" para no perder información. Este proceso se llama promoción de tipos.

Jerarquía de promoción automática

char
→
int
→
long
→
float
→
double

El tipo más pequeño se convierte automáticamente al más grande en la misma operación.

C++ · Conversiones
conversiones.cpp 
int    a = 7;
double b = 2.0;
double resultado;

// Conversión implícita: int → double automáticamente
resultado = a + b;      // a se convierte a 7.0, resultado = 9.0
resultado = a / b;      // a → 7.0, resultado = 3.5 ✅
resultado = a / 2;      // ¡AMBOS son int! resultado = 3.0 ⚠

// Conversión explícita (cast) — la forma correcta
resultado = (double)a / 2;   // Fuerza a a ser double → 3.5 ✅
resultado = static_cast<double>(a) / 2; // C++ moderno, más explícito

// Pérdida de información — narrowing (¡peligroso!)
int entero = (int)3.99;  // entero = 3, NO redondea — trunca
