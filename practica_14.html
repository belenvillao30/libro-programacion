Analizador de Texto — Estadísticas y Validación

string · cctype · conteo de caracteres · palíndromo · validación de cédula · formateo de nombres · estadísticas

Duración180 min (3h)
ModalidadIndividual → Parejas
EntregableCódigo .cpp + tabla de pruebas + bitácora
F1 · Diseño F2 · Pseudocódigo F3 · C++ Individual F4 · Extensión F5 · Análisis IA F6 · Bitácora
Objetivo de la Sesión
El estudiante construye un analizador de texto modular con menú que incluye conteo de categorías de caracteres, detección de palíndromos ignorando mayúsculas y espacios, validación de cédula ecuatoriana, formateo de nombres propios y cifrado César simple. Cada operación es una función con const string& para entradas de solo lectura.
🔤 Contexto del Problema
El laboratorio de Lingüística Computacional de ESPAM MFL desarrolla herramientas de procesamiento de texto en español. Sistema con menú do-while: 1) Estadísticas de texto (letras, dígitos, espacios, otros, porcentajes) — 2) Verificar palíndromo (ignorando mayúsculas, espacios y signos) — 3) Validar cédula ecuatoriana (10 dígitos, provincia 01-24) — 4) Formatear nombre propio (primera letra de cada palabra en mayúscula) — 5) Cifrar texto con César — 6) Salir.
F1
Diseño en Papel — Casos de Prueba y Lógica
0 – 20 min · SIN computadora
1
Casos de prueba para cada función
Diseña en papel al menos 3 casos válidos y 2 inválidos para cada función. Para la cédula: cédulas válidas e inválidas con razón explícita. Para palíndromo: "radar" (sí), "Anita lava la tina" (sí, con espacios), "hello" (no). Para formateo: " juan CARLOS " → "Juan Carlos". Estos casos serán tu lista de verificación en F3.
2
Lógica del cifrado César
El cifrado César desplaza cada letra N posiciones en el alfabeto. Escribe en papel: "ESPAM" con desplazamiento 3 → "HSDPQ". Responde: ¿qué ocurre cuando llegas a 'Z'? ¿Cómo manejas el desbordamiento? ¿Cómo cifras solo letras y dejas los demás caracteres sin cambio?
3
Firmas con const string&
Diseña la firma de cada función. Decide para cada parámetro: ¿la función modifica el string o solo lo lee? Si solo lo lee → const string&. Si necesita una copia modificable → string por valor. Justifica explícitamente cada decisión en tu hoja de diseño.

#include <string>
#include <cctype>
using namespace std;

void   estadisticasTexto(const string& s);
bool   esPalindromo(const string& s);
bool   validarCedula(const string& c);
string formatearNombre(const string& s);
string cifrarCesar(const string& s, int d);

int main() {
    int op; string texto;
    do {
        cout<<"
1.Estadísticas 2.Palíndromo 3.Cédula 4.Nombre 5.César 6.Salir
Opción: ";

        if(op<1||op>6){cout<<"Inválido.
";continue;}

            case 1: estadisticasTexto(texto); break;
            case 2: cout<<(esPalindromo(texto)?"✓ ES palíndromo

"); break;
            case 3: cout<<(validarCedula(texto)?"✓ Cédula válida

            case 4: cout<<"Formateado: "<<formatearNombre(texto)<<"

void estadisticasTexto(const string& s) {
    int letras=0, digitos=0, espacios=0, otros=0;
    for(int i=0;i<(int)s.length();i++) {
        if(isalpha(s[i]))      letras++;
        else if(isdigit(s[i])) digitos++;
        else if(isspace(s[i])) espacios++;
        else                   otros++;
    }
    int total=s.length();
    cout<<"  Total:"<<total<<" | Letras:"<<letras
        <<" | Dígitos:"<<digitos<<" | Espacios:"<<espacios
        <<" | Otros:"<<otros<<"
";
}

bool esPalindromo(const string& s) {
    string limpia;
    for(int i=0;i<(int)s.length();i++)
        if(isalpha(s[i])) limpia+= tolower(s[i]);
    int ini=0, fin=(int)limpia.length()-1;
    while(ini<fin) if(limpia[ini++]!=limpia[fin--]) return false;
    return true;
}

bool validarCedula(const string& c) {
    if((int)c.length()!=10) return false;
    for(int i=0;i<10;i++) if(!isdigit(c[i])) return false;
    int prov = (c[0]-'0')*10 + (c[1]-'0');
    return prov >= 1 && prov <= 24;
}

string formatearNombre(const string& s) {
    string res;
    bool nuevaPal = true;
    for(int i=0;i<(int)s.length();i++) {
        if(isspace(s[i])) { nuevaPal=true; res+=s[i]; }
        else if(nuevaPal) { res+=toupper(s[i]); nuevaPal=false; }
        else res+=tolower(s[i]);
    }
    return res;
}

string cifrarCesar(const string& s, int d) {
    string res;
    for(int i=0;i<(int)s.length();i++) {
        char c=s[i];
        if(isupper(c))      res+= (char)('A'+(c-'A'+d)%26);
        else if(islower(c)) res+= (char)('a'+(c-'a'+d)%26);
        else                res+= c;
    }
    return res;
}
